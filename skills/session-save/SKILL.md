# /session-save 命令使用说明
## 功能
自动将当前会话中的用户提问和Claude回答追加保存到markdown文件中，用于记录学习笔记、项目对话历史。

## 保存规则
1. **文件名**：自动使用当前工作目录的项目名 + `-learn.md`，例如当前在`nanobot`项目则保存为`nanobot-learn.md`
2. **保存格式**：严格按照以下格式追加：
```markdown
## user

```text
用户的问题原文
```

## claude

Claude的完整回答原文
```
3. **文件位置**：保存在当前工作目录根目录
4. **追加模式**：每次调用`/session-save`都在现有文件末尾追加最新的对话，不会覆盖历史内容

## 执行步骤
1. 提取当前会话中最新的一轮或多轮对话（从上次保存后到现在的所有对话）
2. 提取当前工作目录的文件夹名作为项目名
3. 生成保存的markdown内容，严格按照上述格式
4. 检查当前目录是否存在`{项目名}-learn.md`文件
   - 如果不存在：创建新文件，写入内容
   - 如果存在：在文件末尾追加新内容
5. 完成后提示用户保存成功，显示文件路径和本次保存的对话条数
